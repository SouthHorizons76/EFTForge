# v1.1.7 - Community Build Images & Update Checker

**Released:** 2026-04-15

v1.1.7 delivers on the promise from v1.1.5. All community builds will now migrate to use the auto-generated composite gun images. This update also includes an update checker and several build preview improvements.

---

## Community Build Images

All community builds now use the auto-generated composite gun images instead of the factory static images, existing community builds will be automatically migrated to use the new system by the backend server overtime. Images are generated automatically in the background and stored permanently, so community build cards will always show an accurate visual of the full configuration.

New builds submitted from this version onwards have their image generated immediately upon publishing.

---

## Update Checker

The site now checks for new versions automatically. When a new update is available, a notification appears so you know to refresh and get the latest.

---

## Build Preview Improvements

- **Queue overlay** - thanks to community feedback, now when the image generation server is busy handling another request, a non-distracting visual indicator will appear on the gun images instead of the obnoxious toast popping up every time.
- **Abort on navigation** - switching guns or clearing the build now correctly cancels any in-flight generation request, preventing stale images from loading after the build changes
- **Publish confirmation preview** - the confirmation panel before publishing now shows the generated preview image of your current build instead of the factory gun image

---

- Potential fix: Fixed panel width localStorage corruption in no-gun mode
