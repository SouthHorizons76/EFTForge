# v0.9.2 - Load Build Button & News Thumbnails

**Released:** 2026-03-18

v0.9.2 adds a dedicated "Load Build" button to the build editor and improves news card thumbnail handling.

---

## Load Build Button

A fifth button has been added to the build editor toolbar.

Clicking **Load Build** opens the saved builds panel pre-filtered to only show builds saved for the **currently selected weapon**, no need to click the top right Builds button and then find builds for the currently selected gun manually. The search bar inside the panel also searches within that weapon's builds only.

The full builds library (all weapons) remains accessible via the **Builds** button in the header as before.

---

## News Card Custom Thumbnails

News posts can now define an explicit `thumbnail` field in the manifest to use a specific image as the card thumbnail, independent of the post's `title_media`. This allows the hero banner and the list card thumbnail to show different images, and supports per-image positioning and styling overrides via `thumbnail_style` and `thumbnail_img_style`.
