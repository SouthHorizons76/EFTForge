## <span style="color:#f5c542;">EFTForge</span> 的起源

这个项目最初是想要纯抄 [db4tarkov](https://db4tarkov.com) 的，之前这个项目甚至叫做 `db4tarkov CN`，后来才改名为 <span style="color:#f5c542;">EFTForge</span>。我的网站专为中文玩家群体设计，将 UI/UX 和网站性能作为**首要优先级**。想必用过 [db4tarkov](https://db4tarkov.com) 的伙伴都清楚知道她的 UI 设计有多少槽点，而且她的中文本地化也几乎等于没有。

早期版本的 **<span style="color:#f5c542;">EFTForge</span>** 是 `dogshit="true"`.

![EFTForge v0.0.7, 2026-02-20](./news/images/ee/EFTForgev0.0.7.jpg "EFTForge v0.0.7, 2026-02-20")

那时候枪械数据是硬编码的，EvoErgo 公式是从我 ***[已编辑]*** 里掏出来的。

但至少能用 API 获取所有枪械了。

随着时间推移，项目 API 对接变得成熟，计算功能越来越智能，EvoErgo 公式也经过了重写。UI/UX 从"能用"进化成了我引以为傲的东西。网站的每一个元素都是我从零搭起来的，借助了 AI 的帮助 _（是的我vibe code，有意见请你去散步）_。到了 v0.7.0，中文本地化终于上线。我终于有信心录了一个完整的功能演示视频发布到了哔哩哔哩：

## 玩具枪

**20x1毫米玩具手枪**是逃离塔科夫里的一把武器。它发射塑料飞盘，伤害是 1（壹）点，没有任何有意义的改装树。

它也完全没有理由出现在一个武器改装工具里。

**但她被置顶在列表最顶端。** _（这是故意的，不是 bug）_

我这样做因为她给了网站独特的灵魂和个性。不那么商业化舆一本正经。
