# v1.0.7 - Underbarrel Grenade Launcher Improvements

**Released:** 2026-04-02

v1.0.7 improves the current underbarrel grenade launcher system. Weapons with an installed UBGL now show a dedicated grenade ammo selector, include grenade weight in all stat calculations, and save your grenade selection inside builds.

---

## UBGL Ammo Selector

When a UBGL is installed on your weapon, a **UBGL Ammo** row now appears in the stats panel below the regular ammo row. It works the same way as the main ammo selector:

- Lists all compatible grenades for the installed UBGL's caliber, sorted by weight
- Grenade weight is added to total weapon weight (one round per UBGL)
- Weight-dependent stats (EED delta, overswing, arm stamina) are recomputed to reflect the loaded grenade
- Grenade cost is included in the price overview
- Your selection is saved per caliber and restored automatically when the same UBGL is re-equipped
- Ammo selectors are now disabled when "Assume Full Magazine" is toggled off

---

## Build Code UBGL Ammo

UBGL grenade selections are now encoded in build share codes and saved builds. Loading a build that had a grenade selected will restore it automatically. Existing build codes without a grenade selection continue to work unchanged.

---

## Bug Fixes

Weight-dependent stat recomputation (EED, overswing, arm stamina) was previously always recalculated after the ammo weight block even when no ammo weight was actually added - e.g. when no magazine was installed. This is now correctly gated so recomputation only runs when weight was actually changed.

Fixed the stat bars not animating correctly if any one of the three stats goes to the negatives.
