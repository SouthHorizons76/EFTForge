# v0.8.6 - Mobile Compatibility, Panel Resizing & QoL Improvements

**Released:** 2026-03-18

This update brings mobile device support, a draggable panel resizer for desktop, and a bunch of quality-of-life improvements across the site!

---

## Mobile Compatibility

v0.8.6 introduces a temporary mobile layout overhaul designed for touch devices in both portrait and landscape orientations (landscape layout still WIP).

- **Viewport scaling** - Added the correct `viewport` meta tag so the site scales properly on all mobile screens instead of rendering as a shrunken desktop page.
- **Stacked layout with tab bar** - In portrait mode, the gun selector and build tree are stacked vertically with a **Build / Attachments** tab bar at the bottom to switch between them.
- **Landscape mode** - Single-panel tab layout at full screen width with a compact header to maximise vertical space.
- **Long-press to remove** - Long-pressing an installed attachment removes it, matching the right-click behavior on desktop.
- **Tap confirmation modal** - Tapping an attachment opens a confirmation modal showing the attachment's full stats before you install or remove it, since hovering isn't possible on touch devices.
- **Hidden stat columns** - The attachment table hides individual stat columns on mobile to avoid horizontal overflow. All stats remain accessible inside the confirmation modal.
- **2×2 action button grid** - The Back / Reset / Strip / Save buttons are arranged in a 2×2 grid on mobile instead of a horizontal row.
- **Stats panel height cap** - The stats panel is capped in height so the build tree below it stays reachable even when stats are fully expanded.

> Mobile support is new - I am aware that the landscape experience for some mobile devices is still horrible, please report any bugs you encounter.

---

## Manual Panel Resizing

You can now drag the divider between the gun list panel and the build panel to resize them to your preference. The panel sizes are saved and restored automatically across sessions.

---

## Quality of Life

- **Sort button animations** - Clicking any of the sorting buttons (By Caliber / By Class) now plays the gun card entrance animation, making the re-sort feel more responsive.
- **Sort button hover effects** - Added mouse hover styling to the sorting buttons for visual consistency with other interactive elements.
- **Ammo type persistence** - Your selected ammo type is now remembered when you switch between guns or reload the page.
- **Strength level persistence** - Your chosen character strength level is now saved across sessions.
- **Marquee scrolling for saved builds** - Long gun names in the saved builds panel now scroll via a marquee animation instead of being truncated.
- **Dynamic marquee width** - The marquee scroll width and attachment name column width in the attachment table update automatically when the browser window is resized.
- **Server resync warning** - A toast notification now appears as a heads-up before the nightly server data resync at 4:00 AM UTC+8.
