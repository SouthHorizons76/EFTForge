# v1.1.5 - Live Build Preview

**Released:** 2026-04-13

v1.1.5 introduces the Live Build Preview! A real-time composite gun image that reflects your current build as you add and remove attachments, powered by [image-gen.tarkov-changes.com](https://image-gen.tarkov-changes.com/build).

---

## Live Build Preview

The right side placeholder panel (when no slot is clicked into), and the gun image slot in grid view now shows a **live generated image** of your current build. As you modify attachments, the image updates automatically to reflect the full configuration.

Generation is handled by a backend proxy powered by [image-gen.tarkov-changes.com](https://image-gen.tarkov-changes.com/build), and results are cached server-side so repeated builds load instantly.

- **Factory configs** and **bare gun builds** skip generation entirely and use the 512px static images fetched from tarkov.dev directly, so those should have no wait, and send no generation request to put off some server load. naisu😃👍!
- While a new image is generating, the current image dims slightly to indicate an update is in progress. The image only un-dims once the new version has fully loaded.
- If the image generation service is unavailable, a toast notification appears and the panel falls back to the static tarkov.dev image.
- Generated images are cached server-side (up to 500 entries) so the same build configuration loads instantly on repeat visits.

---

## Preview Toggle

A new **Preview** toggle sits to the left of the list/grid view switcher. Turning it off disables image generation entirely and restores the standard static images from tarkov.dev, useful if the generation service is slow or unavailable to avoid the gun images constantly dimming.

The toggle state is saved to your browser so it persists between sessions.

---

## Attachment Table Close Button

The attachment table now has an **X button** in its header to close it without having to right click on an opened slot. Closing the table restores the placeholder panel containing the gun preview image, letting you see the generated composite image while reviewing your build.

---

## Misc

- Increased placeholder panel image size for better build visibility

- A migration of all community builds to use the new generated image preview will be implemented in a later patch, right now, they will still use the factory image except for the featured builds.

- As this is the first iteration of the new generation system, it might be slow or sometimes not work at all. Under the current system, the server can only generate preview images for one user at a time, so when multiple users are using the website at the same time, simultaneous preview image generations may need to be queued. Any feedback is welcome and you can send a bug report through the About panel or you can also DM me on Bilibili!
