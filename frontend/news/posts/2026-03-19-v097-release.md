# v0.9.7 - Gun Card Effects & Stat Change Tracking

**Released:** 2026-03-19

v0.9.7 brings a revamped gun card hover experience, a backend stat change tracking system, and a handful of quality-of-life improvements across the board.

---

## Gun Card 3D Tilt Effect

Gun cards now mimic peeking into a window, with the cards responding to your cursor with a 3D tilt and spotlight glow. As you hover, each card tilts toward your mouse position using a perspective transform, with the weapon image drifting in the same direction at a slower rate for a parallax depth effect.

---

## Attachment Table Slide-In

Opening an attachment slot now plays a short slide-in animation on the attachment table, giving the modding experience a more polished feel.

---

## Gun Search Preserved Across Filters and Language Switch

The gun search field now stays active when switching between Primary/Handgun or Class/Caliber sort, or when the language is switched.

---

## Backend: Stat Change Tracker

The backend now logs weapon and attachment stat changes on every resync. If BSG secretly changed stats of guns or attachments, the system detects the difference and writes it to a `stat_change_log` table with the item name, stat name, old value, new value, timestamp, and whether the sync was scheduled or manual.

The snapshot persists to the server disk across database resets, so the daily resync correctly diffs against the previous day's data even though the database is wiped and rebuilt fresh each time.

This is backend-only for now, a frontend panel showing before/after stat comparisons is planned for a future update when BSG decides to change the stats of items again.

---

## Other Changes

- Fonts converted from OTF to WOFF2 for faster load times
- Load Build dialog now shows the current gun name in the header
- `/traders` API endpoint added, in preparation for the planned trader price update.
- Backend connection failure now auto-retries every 5 seconds instead of showing a one-time error
- Added an easter egg
