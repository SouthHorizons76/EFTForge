# v1.1.3 - Stat Tracker & Sighting Range

**Released:** 2026-04-11

v1.1.3 adds the Stat Tracker! a new panel that surfaces hidden item stat changes detected by the server, along with sighting range in the stats panel, and several QoL improvements & bug fixes.

---

## Stat Tracker

A new **Tracker** button appears in the header. Clicking it opens a side drawer listing any item stat changes detected by the server over the past 7 days.

Each entry shows the item icon, the stat that changed, the old and new values, and the percentage change. Entries are grouped by date.

The server records changes automatically during its daily data sync. If BSG silently nerfs or buffs something, it will show up there.

---

## Sighting Range

The stats panel now shows **Sighting Range** below the other lower stats. It reflects the effective sighting range for your current build, which is the highest sighting range among all installed attachments, falling back to the weapon's base sighting range if none are equipped.

---

## Bug Fixes & Misc

- Fixed the price panel becoming unscrollable when content overflowed
- Fixed AXMC front and rear sight slot overlapping in grid view
- Left panel now clamps and animates back when the browser window is resized smaller than it can accommodate
- In grid view, the attachment table header now shows the installed item's short name on its iconLink
- EvoErgoDelta moved out of the deprecated stat group, now only OverSwing shows as deprecated and the hint displays inline rather than in the old bracket group layout
- Removed the grid view beta feedback hint
- Improved zh localizations
